# Integration Logos

This directory should contain official brand logos for integrations.

## Download Links

1. **Make.com**
   - Source: https://www.make.com/en/brand-assets
   - Files needed: `make.svg` or `make.png` (monochrome variant recommended)

2. **Zapier**
   - Source: https://zapier.com/brand
   - Files needed: `zapier.svg` or `zapier.png` (orange/black variant)

3. **HubSpot**
   - Source: https://www.hubspot.com/company/press
   - Files needed: `hubspot.svg` or `hubspot.png` (from Brand Assets section)

4. **Salesforce**
   - Source: https://www.salesforce.com/company/news-press/brand-assets/
   - Files needed: `salesforce.svg` or `salesforce.png` (blue variant)

5. **Slack**
   - Source: https://slack.com/media-kit
   - Files needed: `slack.svg` or `slack.png` (app icon or logo)

6. **Notion**
   - Source: https://www.notion.so/press
   - Files needed: `notion.svg` or `notion.png` (black-and-white variant)

## Usage

The code will automatically use these local files if they exist, otherwise it will fall back to Simple Icons CDN.

## Specifications

- Logo height: 40-50px
- Format: SVG preferred, PNG acceptable
- Style: Monochrome (white or dark gray) for clean aesthetic
- Alt text: "[Brand] integration logo"
